
FasalAce - Sustainable Farming Frontend

This project is the frontend for the FasalAce application, a Smart India Hackathon 2024 initiative that analyzes soil properties and provides tailored fertilizer recommendations to help farmers optimize crop yield in a sustainable way.

## Project Structure:

1. **fasalace_project.html** - The main HTML file containing the structure of the web application.
2. **styles.css** - The CSS file that provides a clean, greenish theme and styles for the web pages.
3. **script.js** - JavaScript file for client-side functionality, including soil analysis simulation and dynamic weather updates.

## Features:

- **Soil Analysis**: Form input for pH, moisture, and texture data, with output analysis and crop recommendations.
- **Fertilizer Recommendations**: Provides suggestions based on soil conditions, prioritizing sustainable and homemade solutions.
- **Weather Updates**: Simple weather display (mock data for demo purposes).
- **Responsive Design**: The website is mobile-friendly, ensuring accessibility for farmers in remote areas.

## How to Run:

1. Download all the project files:
   - fasalace_project.html
   - styles.css
   - script.js
2. Open the `fasalace_project.html` file in a web browser to view the application.

## Technologies Used:

- HTML5
- CSS3
- JavaScript (for client-side interaction)
